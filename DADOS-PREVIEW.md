## O QUE MUDA NO PREVIEW

| O que muda | O que fica igual (modelo Lovisa) |
|------------|----------------------------------|
| Nome da agência | Pacotes e destinos |
| Logo | Fotos dos destinos |
| WhatsApp | Cores (azul + laranja) |
| Cidade | Depoimentos |
| Instagram @ | Textos do hero |
| URL do Vercel | Seção "Sobre" |

> O preview é uma vitrine para o cliente ver o potencial.
> O conteúdo real será personalizado somente após ele decidir comprar.
